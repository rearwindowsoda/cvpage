DoggerApp Demo

Aplikacja pozwala właściwielom psów na udostępnianie zdjęć i głosowanie najlepszego posta (krótki opis + zdjęcie). Każdy użytkownik ma do dyspozycji jeden głos, który może zmieniać dowolną ilość razy. Psy na które głos oddano najczęściej trafiają na listę TOP10 (aktualny stan w czasie rzeczywistym).

Wykorzystano następujące technologie: HTML, CSS, JS, React, Node.js, Express.js, TypeORM (MySQL), JSON Web Token, Github

Link do aplikacji: owczarek.dev
Link do repozytorium GH (Frontend) github.com/rearwindowsoda/doggerappfront
Link do repozytorium GH (Backend) github.com/rearwindowsoda/doggerapp

ASR Licznik Demo

Aplikacja powstała po to, by ułatwić pracownikom firmy, w której obecnie pracuję, naliczanie sobie punktów za opracowane segmenty radiowe. Pracownicy mają możliwość zapisywać swoje punkty lokalnie do pamięci przeglądarki lub mogą też założyć konto i zapisywać punkty do bazy danych.

Wykorzystano następujące technologie: HTML, CSS, JS, Handlebars, Node.js, Express.js, MongoDB, JSON Web Token, Github

Link do aplikacji: asrlicznik.ct8.pl
Link do repozytorium GH github.com/rearwindowsoda/asr-public-git
